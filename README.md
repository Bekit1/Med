# Family Health — Семейное здоровье

Веб-приложение для отслеживания здоровья всей семьи с AI-ассистентом на базе Claude.

## Возможности

- **Медицинские записи** — анализы, визиты к врачу, рецепты, вакцинации, заметки
- **AI-анализ** — автоматический разбор результатов анализов (PDF, фото) с помощью Claude
- **AI-чат** — медицинский ассистент с доступом к истории здоровья
- **Дашборд** — сводка по здоровью, графики метрик, AI-рекомендации
- **Семейный доступ** — администратор может приглашать членов семьи и просматривать их данные
- **Загрузка файлов** — drag-and-drop для PDF, JPG, PNG, HEIC

## Технологии

- **Next.js 14** (App Router)
- **React 18** + TypeScript
- **Supabase** — PostgreSQL, аутентификация, хранилище файлов
- **Claude AI** (Anthropic) — анализ медицинских документов и чат
- **Tailwind CSS** — тёмная тема
- **Recharts** — графики метрик здоровья

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone <repo-url>
cd Med

# 2. Установить зависимости
npm install

# 3. Настроить переменные окружения
cp .env.local.example .env.local
# Заполните .env.local реальными значениями (см. SETUP.md)

# 4. Запустить dev-сервер
npm run dev
```

Приложение будет доступно на http://localhost:3000

## Деплой на Vercel

1. Push проект на GitHub
2. Откройте [vercel.com](https://vercel.com) → Import Git Repository
3. Добавьте переменные окружения из `.env.local`
4. Нажмите **Deploy**
5. Обновите `NEXT_PUBLIC_SITE_URL` на URL вашего деплоя
